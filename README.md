# Nifty Meek Platform

A fullstack personal operating system built with:

- Next.js (App Router)
- Supabase (Auth + Database)
- Tailwind CSS
- Framer Motion

## Features

- Public brand website
- Admin dashboard (auth protected)
- Full CRUD system
- Build logs timeline (Markdown supported)
- Real-time data rendering

## Philosophy

This project documents real execution:
building, testing, failing, and learning in public.
